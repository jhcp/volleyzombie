Volley

Zombie







Game designed and programmed by João Pimentel, as an entry for The Liberated Pixel Cup (LPC).





This game was developed with HTML, Javascript and CSS, using the Crafty javascript framework.




All images used in this game are from, or derived from:

- the base assets provided by LPC;

- art submitted as an entry to the Art phase of LPC.





The fonts used in this game are 'Courgette', 'Creepster' and 'Press Start 2P', available at Google Fonts.





Music and sounds are from the the Open Game Art repository .





For detailed information about the assets used in this game, please check AUTHORS.TXT.
